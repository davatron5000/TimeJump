TIME​JUMP

Deep-links for Podcasts

Fork me on GitHub

TimeJump adds deep-linking capabilities to HTML5 audio and video podcasts. It works behind-the-scenes to create a standardized API for seeking, based off the YouTube deep-linking syntax.

How it works

TimeJump sniffs out a t (time) query parameter. If it the query parameter, TimeJump will attempt to fast-forward the podcast to the specified timeframe. Then just add one of the following supported query strings to your episode page:

http://mypodca.st/2004/09/first-episode/?t=1h23m45s
http://mypodca.st/2004/09/first-episode/?t=1m23s
http://mypodca.st/2004/09/first-episode/?t=1:23
